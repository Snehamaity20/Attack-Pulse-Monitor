# Attack Pulse Monitor

A Python-based network monitoring tool that captures live packets and detects common network attacks in real time using **Scapy**.

## Features

* Live packet sniffing
* Real-time dashboard
* TCP, UDP, ICMP, and ARP monitoring
* ICMP Flood detection
* DDoS detection
* ARP Spoofing detection
* Live traffic and alert logging

## Technologies

* Python
* Scapy
* Colorama
* PyFiglet

## Installation

```bash
pip install scapy colorama pyfiglet
```

## Run

```bash
sudo python3 attack_pulse_monitor.py
```
